# spacex-timeline

Hello, this is a small project built with VueJS, Vuetify and the free spaceX API (https://github.com/r-spacex/SpaceX-API)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
